## 使用示例

### 快速开始

```bash
# 1. 静态分析（快速）
python scripts/deep_protection.py core/agent.py --no-agent

# 2. Agent 审查（深度）
python scripts/agent_review.py core/agent.py

# 3. CLI 审查
python cli.py
/review core/agent.py
```

### 完整项目检查

```bash
# 检查所有 Python 文件
python scripts/deep_protection.py --all --json > report.json

# 或使用 CLI
python cli.py
/review --all
```

### 生成报告

```bash
# 文本报告
python scripts/deep_protection.py core/ -v > report.txt

# JSON 报告
python scripts/deep_protection.py core/ --json > report.json

# Agent 审查报告
python scripts/agent_review.py core/ -o review.txt
```
